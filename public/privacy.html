What This App Does

  • Create, manage, and run exams.
  • Provide exam-level analysis, subject-level analysis, and user-level analysis.
  • Support AI-assisted generation and evaluation workflows.

Data We Collect

Account and Profile

  • Google account information: googleId, email, display name, photo.
  • App profile and role data (teacher or student context).

Exam and Learning Data

  • Subjects, exams, questions, answers, scores, and analytics outputs.
  • Uploaded educational files and text used for exam generation.
  • Personal assessment context for student progress.

Device and Session Data

  • Device metadata from Android/iOS to identify login sessions.
  • Session/authentication tokens for secure API access.

Device Identification and Token Control

To control sessions and reduce uncontrolled token spread, we collect structured device info during sign-in.

  • Android: manufacturer, model, brand, OS version, device, product, and Android ID.
  • iOS: model, OS version, name, system name, and identifierForVendor.
  • Policy: maximum active login devices per account is capped at 3 devices.

How We Use Data

  • Authentication and account management.
  • Running exams securely and calculating results.
  • Producing analytics at exam, subject, and user levels.
  • Improving quality and ease of use.
  • Operating AI flows with model fallback behavior.

AI Processing

  • The app can use more than one Gemini model.
  • If one model is unavailable or rate-limited, processing may continue with the next configured model.

Sharing and Disclosure

  • We do not sell personal data.
  • We do not share personal data with advertisers for profiling.
  • Data is shared only with required infrastructure/services (for example, auth, storage, notifications, AI provider APIs) and only as needed to run the app.

Security

  • Token-based authenticated API calls.
  • Secure transport for backend communication.
  • Exam integrity mechanisms (including anti-cheat behavior in exam flows).

Retention and Deletion

  • We retain data while needed for app functionality and educational records.
  • You can request account/data deletion from app profile actions or by contacting support.

Planned Product Expansion

Future modules may include plans/subscriptions, teacher workshops, ads support, and multi-language support. If data usage changes, this policy will be updated.
